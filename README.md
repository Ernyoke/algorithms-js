## Algorithms:

* Graphs:
    * Undirected Graphs:
        * Depth-First Search (DFS)
        * Breath-First Search (BFS)
        * Minimum Trees:
            * Prim's
            * Kruskal
        * Shortest Path:
            * Dijkstra's
        
    * Directed Graphs
        * Depth-First Search (DFS)
        * Breath-First Search (BFS)
        * Minimum Trees:
            * Prim's
            * Kruskal

* Search
    * Binary Search (recursive and non-recursive)
    * Find If

* Sort
    * Quick-sort
    * Insertion sort
    
* Arrays:
    * Reverse
    * Rotate
    * Partition
    * Partial Sum
    * Utilities (bound validation, element swapping)

* Mathematical
    * Is Power Of 2
    * Square Root - Newton-Raphson method
    * Is Number Prime
    
* Dynamic Programming
    * Fibonacci

* Miscellaneous
    * Is Parenthesis Order Balanced
    * Sort By Binary Ones
